# Hour hearer app

## Based on [TailAdmin React](https://free-react-demo.tailadmin.com/)

You'll need to install Node.js >=v14.16+ and TailAdmin uses **Vite** for frontend tooling, to peform installation and building production version, please follow these steps from below:

- Use terminal and navigate to the project (tailadmin-react) root.
- Then run : <code>npm install</code>
- Then run : <code>npm run dev</code>

**For Production Build**
Run : <code>npm run build</code>

Default build output directory: /dist
